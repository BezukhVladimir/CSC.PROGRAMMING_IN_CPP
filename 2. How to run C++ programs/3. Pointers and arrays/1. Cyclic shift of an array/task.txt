<p>В этой задаче вам нужно реализовать функцию, которая сдвигает содержимое массива <b>влево</b> на заданное число позиций (циклический сдвиг). </p><p>На вход функция принимает массив, его размер и величину сдвига. Например, если на вход функции подан массив: int a[] = { 1, 2, 3, 4, 5 }; и требуется циклически сдвинуть его влево на 2 позиции, то на выходе мы получим числа в таком порядке: 3, 4, 5, 1, 2.</p><p>Обратите внимание, что величина сдвига может быть нулевой, а может быть и больше размера массива, все эти случаи нужно учесть.</p><p><b>Требования к реализации: </b>вам нужно реализовать только функцию циклического сдвига. При этом она не должна вводить или выводить что-либо. При решении этой задачи вы можете определять любые вспомогательные функции, если они вам нужны. Реализовывать функцию main не нужно. Предполагается, что вам не потребуется дополнительная память, а именно: массивы и стандартные контейнеры. Пользоваться стандартными алгоритмами и контейнерами не разрешается, даже если вы с ними знакомы.</p><p><b>Рекомендация: </b>перед тем, как начать кодировать решение этой задачи, продумайте алгоритм, который вы будете использовать. В данной задаче не проверяется эффективность этого алгоритма (в разумных пределах).</p><p><b>Подсказка: </b>вам может потребоваться оператор % для вычисления остатка. Наиболее <i>простая</i> реализация этой функции сдвигает массив на один элемент за проход. Наиболее <i>простая</i> реализация несколько раз использует функцию, которая переставляет элементы массива в обратном порядке. </p><br><br><font color="gray">Memory limit: 32 MB</font><br><font color="gray">Time limit: 1 second</font><br><br>

<p>Напишите программу для вычисления целочисленного логарифма по основанию 2. На вход программе в первой строке подается число <span class="math-tex">\(  T  \)</span>, далее следует <span class="math-tex">\(  T  \)</span> строк с тестами. Каждый тест состоит из одного целого положительного числа <span class="math-tex">\(  a_i \lt  10^{9} \)</span>, для каждого <span class="math-tex">\(  a_i  \)</span> нужно вывести на отдельной строке такое наибольшее число <span class="math-tex">\(  p  \)</span>, что <span class="math-tex">\(  2^p \le a_i  \)</span>. Гарантируется, что <span class="math-tex">\(  a_i \ge  1  \)</span> . При решении задачи вы можете определять любые вспомогательные функции, если они вам нужны, более того рекомендуется вынести вычисление логарифма в отдельную функцию.</p><p>Ограничения: библиотеку cmath (и math.h) использовать запрещено.<br></p><br><b>Sample Input:</b><br><pre><code class="language-no-highlight">5<br>59218<br>50960<br>42043<br>42817<br>9998</code></pre><br><b>Sample Output:</b><br><pre><code class="language-no-highlight">15<br>15<br>15<br>15<br>13</code></pre><br><br><br><font color="gray">Memory limit: 32 MB</font><br><font color="gray">Time limit: 1 second</font><br><br>

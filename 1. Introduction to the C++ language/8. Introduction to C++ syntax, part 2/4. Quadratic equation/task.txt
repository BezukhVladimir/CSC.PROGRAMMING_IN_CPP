<p>Напишите программу для решения квадратных уравнений вида <span class="math-tex">\(  a x^2 + b x + c = 0  \)</span> (относительно <span class="math-tex">\(  x  \)</span>). На вход программа получает три целых числа: <span class="math-tex">\(  a  \)</span> , <span class="math-tex">\(  b  \)</span> и <span class="math-tex">\(  c  \)</span>, соответственно. При этом гарантируется, что <span class="math-tex">\(  a \ne 0  \)</span>. На вывод программа должна вывести два вещественных корня уравнения, разделённые пробелом. Если вещественных корней нет, то программа должна вывести строку "No real roots". Если у уравнения имеется только один корень (кратный корень), то программа должна вывести его дважды. Порядок вывода корней не важен. Ничего, кроме этого, выводить не нужно. Для вычислений с плавающей точкой используйте тип double. При выполнении задания вам может оказаться полезной функция sqrt из заголовочного файла cmath.</p><p><i>Замечание:</i> после того, как вы сдадите это задание, попробуйте подумать, как изменилась бы ваша программа, если бы мы не гарантировали, что <span class="math-tex">\( a\neq 0 \)</span>. </p><br><b>Sample Input:</b><br><pre><code class="language-no-highlight">1 4 4</code></pre><br><b>Sample Output:</b><br><pre><code class="language-no-highlight">-2 -2</code></pre><br><br><b>Sample Input:</b><br><pre><code class="language-no-highlight">1 0 1</code></pre><br><b>Sample Output:</b><br><pre><code class="language-no-highlight">No real roots</code></pre><br><br><b>Sample Input:</b><br><pre><code class="language-no-highlight">1 0 0</code></pre><br><b>Sample Output:</b><br><pre><code class="language-no-highlight">0 0</code></pre><br><br><br><font color="gray">Memory limit: 32 MB</font><br><font color="gray">Time limit: 1 second</font><br><br>
